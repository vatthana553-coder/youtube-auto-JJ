# 🎬 Free AI YouTube Automation (Zack D. Films Style)

This guide helps you create viral faceless YouTube Shorts like **Zack D. Films** using 100% free AI tools.

## 🛠️ The Free Tech Stack

| Component | Tool | Cost | Why |
|-----------|------|------|-----|
| **Script Writing** | [Ollama + Llama3](https://ollama.com) or [HuggingChat](https://huggingface.co/chat) | Free | Local LLM for unlimited scripts |
| **Voiceover** | [Edge TTS](https://github.com/rany2/edge-tts) | Free | Microsoft's neural voices (same quality as paid tools) |
| **Images** | [Bing Image Creator](https://bing.com/images/create) or [Leonardo.ai](https://leonardo.ai) | Free | High-quality 3D renders |
| **Editing** | [CapCut Desktop](https://capcut.com) | Free | Auto-captions, keyframes, effects |

## 🚀 Quick Start

### Option A: Use the Python Script (Semi-Automated)

1. **Install Dependencies**:
   ```bash
   pip install edge-tts
   ```

2. **Run the Script**:
   ```bash
   python auto_zack_style.py
   ```

3. **Follow the prompts** to generate voiceover and image prompts.

### Option B: Fully Manual Workflow (No Coding)

#### Step 1: Generate Script
Go to **HuggingChat** or use **Ollama** locally with this prompt:

```
You are a scriptwriter for a viral YouTube Shorts channel like 'Zack D. Films'.
Topic: [YOUR TOPIC]

Rules:
1. Hook in the first 3 seconds.
2. Keep sentences short and punchy (max 15 words per sentence).
3. Total length under 50 seconds (approx 130 words).
4. Tone: Curious, fast-paced, slightly mysterious.
5. For each sentence, provide an image prompt for a 3D renderer.

Output format:
Sentence: "..." 
Image Prompt: "3d render, pixar style, ..."
```

#### Step 2: Generate Voiceover
Use **Edge TTS** (Free):

```bash
# Install
pip install edge-tts

# Generate audio (GuyNeural is similar to Zack D.'s voice)
edge-tts --voice en-US-GuyNeural --text "Your script text here" --write-media voiceover.mp3
```

Or use the online version: [Edge TTS Web](https://www.ttsmaker.com)

#### Step 3: Generate Images
Copy each image prompt into:
- **Bing Image Creator** (Fastest, Free, DALL-E 3 powered)
- **Leonardo.ai** (More control, free daily tokens)

**Pro Tip for Zack D. Style:**
Add these keywords to every prompt:
> `3d render, pixar style, cute characters, bright lighting, cross section view, educational diagram, 8k`

#### Step 4: Edit in CapCut

1. Import `voiceover.mp3`
2. Drag in your generated images
3. **Keyframe Animations** (Crucial!):
   - Select image → Add keyframe at start (scale 100%)
   - Move to end of clip → Scale to 110% (slow zoom)
   - This creates the dynamic "moving image" effect
4. **Auto Captions**:
   - Text → Auto Captions
   - Font: "The Bold Font" or "Anton"
   - Animation: "Spring" or "Pop up"
   - Color: Yellow/White with black stroke
5. **Sound Effects**:
   - Add subtle "whoosh" sounds between transitions
   - Add "pop" sounds when text appears

## 📝 Example Workflow: "What If You Swallowed Gum?"

### 1. Script (Generated by AI)
```
Have you ever wondered what actually happens to gum in your stomach?
Most people think it stays there for seven years.
But doctors say that is a complete lie.
Your body simply moves it through your digestive system normally.
It comes out exactly the same way it went in, just a few days later.
```

### 2. Image Prompts
```
1. "3d render, cute cartoon boy swallowing gum, cross section view of throat, bright lighting, pixar style"
2. "3d render, old calendar flipping rapidly, dust accumulating on gum, dark moody lighting"
3. "3d render, doctor shaking head, big red FALSE text, dramatic studio lighting"
4. "3d render, digestive tract animation, gum moving through intestines, medical illustration"
5. "3d render, toilet flushing, humorous, bright bathroom, pixar style"
```

### 3. Final Assembly
- Voiceover: ~15 seconds
- 5 Images with zoom keyframes
- Bold captions with pop animation
- Background music: Light curious track (YouTube Audio Library)

## 💡 Pro Tips for Viral Success

1. **The Hook**: First 3 seconds must show something weird/shocking
2. **Pacing**: Change image every 2-4 seconds
3. **Color Grade**: Increase saturation in CapCut for vibrant look
4. **Consistency**: Post at same time daily
5. **Trending Topics**: Use Google Trends to find weird facts people search
